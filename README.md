# 📧 MailForge

MailForge is a modern, clean and responsive SaaS landing page built for email marketing platforms.  
It showcases a professional UI for email campaign management, designed for startups, SaaS products, and marketing tools.

---

## 🚀 Live Demo

👉 https://codebyalpesh.github.io/mailforge/

---

## ✨ Features

- ⚡ Modern SaaS Landing Page UI  
- 🎯 Hero section with strong value proposition  
- 🧩 Modular sections (Features, Process, Testimonials, CTA)  
- 📱 Fully responsive design (Mobile + Tablet + Desktop)  
- 🎨 Clean UI with design system approach  
- 🧠 Conversion-focused layout structure  
- 💡 Smooth and professional visual hierarchy  
- 🖼️ Ready for scaling into full SaaS product
